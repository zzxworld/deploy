# deploy

常用的开发工具部署脚本。

## 测试平台:

* CentOS 7

## 使用:

### install rbenv
```
curl -sSL https://raw.githubusercontent.com/zzxworld/deploy/master/rbenv.sh | bash
```

使用镜像 (_Ruby China_):
```
curl -sSL https://raw.githubusercontent.com/zzxworld/deploy/master/rbenv.sh | bash -s https://gems.ruby-china.org
```
